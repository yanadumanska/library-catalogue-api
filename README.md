# Library Catalogue

RESTful API + React для каталогу бібліотеки.

---

## Стек

- **Backend:** Java 17, Spring Boot, PostgreSQL, Maven
- **Frontend:** React 
- **DevOps:** Docker, AWS 

---

## Команда та поділ задач

### Yana Dumanska
- Books (CRUD + 9 фільтрів + пошук + сортування + пагінація)
- Authors (CRUD + пошук)
- Categories (CRUD + ієрархічне дерево)
- Reviews (CRUD + перерахунок рейтингу)
- Caching (ETag, Cache-Control)

### Nadiia Zinchuk
- Users (CRUD + реєстрація)
- JWT (access + refresh токени)
- Borrowings (borrow, return, renew, overdue)
- Reservations (reserve, cancel)
- Admin (управління ролями)
